# Discipline Tracker App

## 🚀 Overview
A group-based discipline tracking application designed to build consistency through accountability, structured tracking, and social pressure.

This app focuses on tracking daily habits such as workouts, nutrition, sleep, and productivity while maintaining a 30-day challenge system among friends.

---

## 🎯 Objective
To help users stay consistent in their daily routines by:
- Tracking key discipline metrics
- Maintaining streaks
- Enforcing accountability through group visibility
- Encouraging progress through structured challenges

---

## 🧩 Core Features (Planned)

### 🔹 Challenge System
- 30-day challenge format
- Fixed start and end date
- Daily tracking

### 🔹 Workout Tracking
- Predefined or custom workout plans
- Exercise-level tracking (sets & reps)
- Completion status (Complete / Partial / Failed)

### 🔹 Discipline Metrics
- Protein intake
- Calorie tracking
- Sleep duration
- Social media usage
- Upskilling time

### 🔹 Streak System
- Daily streak tracking
- Reset on failure
- Minimum completion logic

### 🔹 Group Accountability
- Leaderboard system
- Friend comparison
- Public visibility of performance

### 🔹 Penalty System
- Punishment tasks for missed days
- No monetary penalties

### 🔹 Proof System
- Upload workout photo/video
- Unverified flag if not provided

### 🔹 Weekly Tracking
- Weekly weight updates
- Progress trends

---
